# Transporti V1 - Plateforme de Logistique

Bienvenue dans le dépôt de **Transporti V1**, une solution moderne pour la gestion et l'optimisation du transport de marchandises.

## 📁 Structure du Projet

Ce dépôt est organisé en **Monorepo** :

*   **[`backend/`](./backend/README.md)** : API Django REST Framework. Gère la logique métier, la base de données (PostgreSQL) et l'authentification.
*   **[`frontend/`](./frontend/README.md)** : Application Next.js. Interface utilisateur pour les clients, transporteurs et administrateurs.
*   **[`transportiv1_artifacts/`](./transportiv1_artifacts)** : Documentation technique détaillée et artefacts du projet.

## 🚀 Démarrage Rapide

Pour installer et lancer le projet localement, veuillez consulter les guides spécifiques :

1.  **Démarrer le Backend :** [Guide Backend](./backend/README.md)
2.  **Démarrer le Frontend :** [Guide Frontend](./frontend/README.md)
